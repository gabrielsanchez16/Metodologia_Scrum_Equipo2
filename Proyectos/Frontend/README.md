# 🛠️ Sistema de Gestión de Talleres y Almacenes de Repuestos

Este proyecto es una aplicación web para la **gestión integral de talleres de motocicletas y almacenes de repuestos**.  
El sistema está diseñado para optimizar la administración de órdenes de trabajo, clientes, mecánicos, inventarios y ventas, ofreciendo además funcionalidades de suscripciones y notificaciones en tiempo real.

---

## 🚀 Funcionalidades principales

### 🔧 Gestión de Taller
- Registro de **órdenes de trabajo** para clientes.
- Subida de **evidencias** (imágenes, documentos, etc.).
- Agregar **descripciones detalladas** del servicio.
- Adjuntar **repuestos y consumibles** usados en la orden.
- Seguimiento en tiempo real del estado de la motocicleta en el taller.
- Notificaciones instantáneas para los clientes.

### 📦 Gestión de Almacén
- **Control de inventario** con alertas de bajo stock.
- Gestión de repuestos vinculados a órdenes de trabajo.
- Registro de entradas y salidas de productos.

### 👥 Administración de Usuarios
- Registro y gestión de **clientes**.
- Registro y control de **mecánicos**.
- Registro y administración de **motocicletas** asociadas a cada cliente.

### 💰 Finanzas y Ventas
- Generación de **cotizaciones**.
- Visualización de **ventas mensuales**.
- **Filtros y reportes** detallados por fechas, clientes o servicios.

### 🎟️ Suscripciones y Membresías
- Diferentes niveles de suscripción:
  - **Básico**
  - **Premium**
  - **Profesional/Avanzado**
- Beneficios especiales según el plan:
  - Acceso a más funcionalidades.
  - Publicidad destacada en la página principal.
  - Recomendación de talleres y almacenes de repuestos suscritos al plan más alto.

### 🔐 Autenticación y Seguridad
- Sistema de **login y registro seguro**.
- Roles y permisos según el tipo de usuario (administrador, mecánico, cliente).

---

## 🌟 Beneficios del sistema
- Mejora la **organización interna del taller**.
- Facilita la **comunicación con los clientes** mediante notificaciones en tiempo real.
- Optimiza el **control del inventario** evitando pérdidas o sobrecostos.
- Permite **aumentar la visibilidad de talleres y almacenes** mediante el sistema de suscripciones.
- Genera informes claros de **ventas y desempeño mensual**.

---

## 🛠️ Tecnologías utilizadas
- **Backend:** Nodejs
- **Frontend:** React / Typescript / TailwindCSS
- **Base de datos:** MySQL / Sequelize
- **Autenticación:** Nodejs y express
- **Notificaciones en tiempo real:** SendMsg / Mailtrap
- **Control de versiones:** Git & GitHub
